OceanWise



































O que é o Projeto?

O projeto OceanWise foi desenvolvido com a finalidade de ensinar as crianças e conscientiza-las sobre os desafios do oceano e como podemos contribuir para a preservação desse ambiente tão importante para todos os seres vivos. A iniciativa conta com uma apostila para o primeiro, quinto e nono ano do ensino fundamental contando com atividades e um aprendizado para toda a vida.

As escolas inscritas no programa terão acesso a um certificado via PDF em que poderão escrever o nome de cada estudante e imprimir para ser entregue ao fim do programa como uma lembrança de todo o conhecimento adquirido

Baixar Apostila 1
